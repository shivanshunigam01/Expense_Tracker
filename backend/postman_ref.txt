## üîê AUTH APIs

---

### ‚úÖ 1. **Signup**

**POST** `/auth/signup`

**Request Body** (JSON):
```json
{
  "name": "Shivanshu",
  "email": "shivanshu@example.com",
  "password": "123456"
}
```

**Response:**
```json
{
  "message": "Signup successful"
}
```

---

### ‚úÖ 2. **Login**

**POST** `/auth/login`

**Request Body** (JSON):
```json
{
  "email": "shivanshu@example.com",
  "password": "123456"
}
```

**Response:**
```json
{
  "token": "<JWT_TOKEN>",
  "user": {
    "id": 1,
    "name": "Shivanshu",
    "email": "shivanshu@example.com"
  }
}
```

---

### ‚úÖ 3. **Get Profile**

**GET** `/auth/profile`  
**Header:**  
`Authorization: Bearer <JWT_TOKEN>`

**Response:**
```json
{
  "id": 1,
  "name": "Shivanshu",
  "email": "shivanshu@example.com",
  "status": 1,
  "created_at": "2024-12-01T10:23:45.000Z"
}
```

## üí∏ EXPENSE APIs

---

### ‚úÖ 4. **Add Expense**

**POST** `/expenses`  
**Header:**  
`Authorization: Bearer <JWT_TOKEN>`

**Request Body:**
```json
{
  "category": 1,
  "amount": 250.50,
  "date": "2024-12-05",
  "description": "Lunch"
}
```

**Response:**
```json
{
  "message": "Expense added"
}
```

---

### ‚úÖ 5. **Update Expense**

**PUT** `/expenses/1`  
**Header:**  
`Authorization: Bearer <JWT_TOKEN>`

**Request Body:**
```json
{
  "category": 2,
  "amount": 400.00,
  "date": "2024-12-06",
  "description": "Taxi ride"
}
```

**Response:**
```json
{
  "message": "Expense updated"
}
```

---

### ‚úÖ 6. **Delete Expense**

**DELETE** `/expenses/1`  
**Header:**  
`Authorization: Bearer <JWT_TOKEN>`

**Response:**
```json
{
  "message": "Expense deleted"
}
```

---

### ‚úÖ 7. **Get Filtered Expenses**

**Header:**  
`Authorization: Bearer <JWT_TOKEN>`

**Response:**
```json

    "id": 1,
    "user_id": 1,
    "category": 2,
    "amount": 400.00,
    "date": "2024-12-06",
    "description": "Taxi ride"
  },
  ...
]
```
